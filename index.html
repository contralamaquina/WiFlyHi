Pensado para guardar clases o conversaciones largas: el texto fluye seguido y solo separa en párrafos cuando hay una pausa real. La voz se transcribe con un modelo (Whisper) que corre local, en tu teléfono — el audio nunca sale del dispositivo. La primera vez que transcribís descarga el modelo (~75MB, una sola vez); después queda guardado y funciona incluso sin internet. El corte de cada frase lo maneja este código, no el navegador, así que podés ajustar cuánto silencio hace falta para cortar con el control de arriba.
